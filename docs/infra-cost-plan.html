Premissas de Uso

Padrões de consumo por cliente FIDC
Perfil do usuário: Operador de FIDC — compliance officer, gestor de carteira, analista de risco.

Usa o sistema em horário comercial (8h-18h BRT), com picos às 10h e 15h. Consultas variam de perguntas rápidas ("qual o PDD atual?") a análises complexas ("avalie concentração do cedente X").

O custo real não é a inferência.

Em serverless, o custo dominante é o idle time — os 180s de scaledown window após cada burst de queries. Com poucos clientes, o container fica ocioso 90% do tempo e você paga pela espera.

O cálculo que importa: GPU-hora efetiva

Para N queries/dia com burst pattern (3-5 queries, depois pausa), o container fica ativo por:

tempo_ativo = (N_bursts × 180s) + (N_queries × tempo_por_query) + cold_starts

O paradoxo do self-hosted

API pura (Together/Groq) é 2-3x mais barato que self-hosted para ≤100 clientes. Mas self-hosted dá: controle total, zero vendor lock-in, dados nunca saem da infra, compliance LGPD simplificado, e capacidade de fine-tune.

A estratégia ótima: API first (Phase 1-2), migrar pra self-hosted quando volume justificar ou compliance exigir.

Decisões Estratégicas de Inflexão

Quando mudar de approach
TriggerAçãoPor quê
0→20 clientes Together AI puro Zero infra, zero cold start, pay-per-token. Mais barato e mais confiável que self-hosted para volume baixo.
20→50 clientes Híbrido (API + Modal standby) API primário, Modal como fallback e para queries que exigem dados on-premise (compliance).
50→100 clientes Modal primary Volume justifica containers always-on. Cold starts eliminados. Custo por query cai.
100+ clientes Dedicated GPU cluster Lambda/Vast.ai A10G dedicated ~$350/mês cada. 2-3 GPUs = cluster resiliente.
Cliente enterprise (banco) On-premise deploy Regulação exige dados dentro da infra do banco. Docker + GPU on-prem. Pricing: R$25K/mês.
Latência > 5s (warm) Upgrade GPU (L40S ou H100) Se demanda exceder capacidade do A10G. H100 = 5x mais rápido, 4x mais caro.
Fine-tune pronto Modelo custom < 7B Fine-tune em dados FIDC pode produzir modelo 3B com qualidade equivalente ao 27B generalista. Custo cai 10x.
