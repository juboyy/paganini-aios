🧬 MetaClaw

1 Proxy intercepts LLM call
2 Searches skill library (embedding similarity)
3 Injects top 6 skills into system prompt
4 Post-session: LLM analyzes conversation
5 Auto-generates new skills (markdown)
6 Validates vs corpus (no contradictions)
7 Next session is measurably better
Session 50 > Session 1. No fine-tuning.

🔍 AutoResearch

1 LLM reads program.md (optimization instructions)
2 Reads pipeline.py (current RAG config)
3 Reads experiments.jsonl (past results)
4 Hypothesizes improvement
5 Modifies pipeline.py (16 tunable params)
6 Runs eval.py (50-100 gold Q&A pairs)
7 Improved? Commit. Degraded? Revert.
Continuous retrieval optimization. Autonomous.

🧠 Memory Reflection

1 Daily daemon reviews all operations
2 Extracts patterns from interactions
3 Builds knowledge graph (entities + relations)
4 Promotes episodic → semantic memory
5 Prunes outdated knowledge
6 Detects contradictions
7 All agents benefit next morning
Knowledge compounds. Day 100 >> Day 1.
MetaClaw Modes
From Skill Injection to Weight Updates
Mode 1: skills_only (default)
No GPU. No fine-tuning. Skills injected into prompt at every turn. Auto-generated post-session. Chinese walls per fund. Skill validation against corpus. Max 500 active skills with weekly pruning.
Mode 2: rl (optional)
+ Live LoRA fine-tuning via Tinker Cloud. PRM judge scores every response (0.0-1.0). GRPO optimization. LoRA rank 32. Weights hot-swapped every 4 samples — zero downtime. Failed episodes generate corrective skills.
Mode 3: opd (advanced)
Teacher-student distillation. Frontier model (GPT-5.2) teaches smaller model (Qwen3-7B). Per-token log-probabilities. KL penalty alignment. After N sessions: student ≈ teacher quality at 1/10th cost.
Security
Built In, Not Bolted On

🔒 Container Isolation

Agents run in containers with network: none. Unix socket only. cap-drop ALL. Read-only FS. Distroless. Seccomp. PID limit 50.

🧱 Chinese Walls

Fund A ≠ Fund B at every layer. RLS per fund_id. Partitioned memory. Isolated MetaClaw skills. Per-fund traces.

🔑 Encrypted Vault

AES-256-GCM. No plaintext secrets. Pre-commit hooks scan for keys, PII, corpus fingerprints. CI: trufflehog + gitleaks.

🛡️ 6-Gate Pipeline

Eligibility → Concentration → Covenant → PLD/AML → Compliance → Risk. First BLOCK kills operation. Override = human + audit.

📋 Immutable Audit

Every decision traced with gate_token + fund_id. 7-year retention. Append-only. CVM auditor reconstructs any operation from traces.

🌐 Egress Allowlist

Only whitelisted domains pass: CVM, ANBIMA, BACEN, LLM providers, Slack. All else blocked. Every request logged.
