03 — Fontes de Receita

7 Streams de Receita

De subscription recorrente a marketplace de skills. Cada fonte ativa em um estágio diferente de maturidade do produto.

Ativo

1. Subscription (Packs)

MRR via planos mensais/anuais. Starter gratuito, Professional e Enterprise pagos. Contrato mínimo anual para Enterprise.
Administradora com 10 fundos no plano Enterprise: R$25K/mês. Desconto anual 15% = R$255K/ano.
R$8-25K/cliente/mês
Ativo

2. Usage-Based (Tokens)

Markup de 60% sobre custo de LLM. Cliente paga pelo que consome acima do incluído no plano. BYOK opcional.
Fundo ativo: 200 queries + 20 relatórios + 500K tokens = ~R$130/mês de usage adicional.
R$3-8K/cliente/mês
Ativo

3. Setup / Onboarding

Receita one-time por implementação. Quick Start (1 dia), Standard (1 semana), Enterprise (2-4 semanas). Inclui deploy, corpus ingest, treinamento.
Enterprise onboarding: deploy on-prem + HA, todos os fundos, integrações, treinamento 20h, guardrails customizados.
R$5-50K one-time
3-6 meses

4. Custom Agents

Desenvolvimento sob demanda de agentes especializados. Agent de Cessão de Crédito, Monitoramento de Cedentes, Stress Test, Reconciliação.
Agent de Análise de Cessão: cruza SPC + Serasa + histórico do cedente. Valida lastro, calcula deságio, verifica duplicatas. Integra com CERC/CIP.
R$15-120K/agent
3-6 meses

5. API Licensing

Endpoints REST para integração com sistemas do cliente. Compliance-check, query RAG, report generation, risk assessment, PLD screening.
Fintech processa 2.000 cessões/mês. Cada cessão: compliance-check (R$0.50) + risk-assess (R$1.00) + pld-screen (R$2.00) = R$7K/mês.
R$500-30K/mês
6-12 meses

Desenvolvedores terceiros criam skills especializados (due-diligence-imobiliária, compliance-bacen-pix, auditoria-previdência). Paganini fica com 30%.
Consultoria de CRI cria skill "due-diligence-imobiliária" a R$2K/mês. 10 clientes = R$20K/mês. Share Paganini: R$6K/mês.

08 — Riscos

Riscos e Mitigações

IA não é confiável para compliance

Alta prob.

Humano-no-loop obrigatório. Agente recomenda, humano aprova. Trilha de auditoria completa. POC de 30 dias prova a qualidade.

Regulação CVM muda

Média prob.

Regulatory Watch agent monitora mudanças diariamente em CVM/ANBIMA/BACEN. Update de guardrails em <48h. Corpus re-indexado automaticamente.

LLM provider muda preços

Média prob.

BYOK + suporte a múltiplos providers (OpenAI, Google, Anthropic, Ollama local). Migração em horas. Sem vendor lock-in.

Competidor com mais capital

Média prob.

First-mover advantage. Corpus regulatório brasileiro curado (leva meses para replicar). Open source cria comunidade defensável.

Churn alto nos primeiros meses

Média prob.

Onboarding dedicado. NPS quinzenal. Feature roadmap priorizado por feedback direto dos primeiros clientes.

Vazamento de dados sensíveis

Baixa prob.

Deploy on-prem. Zero data retention. Auditoria trimestral. Chinese walls por fund_id. Seguro cyber. LGPD-compliant by design.
