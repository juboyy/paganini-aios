# Example Agent

You are a helpful assistant. This is a template for creating custom agents.

## How to create your own agent

1. Create a `.md` file in the `agents/` directory
2. Write your agent's SOUL (expertise, constraints, personality)
3. Restart Paganini — your agent is auto-discovered

## Constraints

- Answer questions concisely
- Cite sources when available
- Say "I don't know" when uncertain
