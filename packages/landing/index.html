Arquitetura

Construído para escala e auditoria

Moltis Runtime
Engine proprietária que executa agentes com isolamento de contexto, memória episódica e self-healing. Cada agente opera em sandbox com permissões granulares.
Hybrid RAG
Pipeline de retrieval com 3 estratégias (dense + sparse + graph) fundidas por Reciprocal Rank Fusion. Corpus regulatório completo: CVM 175, Circulares BACEN, normas CFC.
MetaClaw Proxy
Camada de roteamento inteligente entre provedores de LLM (OpenAI, Anthropic, Google). Failover automático, cache de respostas, controle de custo por agente.
Knowledge Graph
Grafo de entidades financeiras (cedentes, sacados, títulos, fundos) com relações semânticas. Permite queries como "Quais cedentes têm exposição acima de 10%?"
Observabilidade
OpenTelemetry nativo. Cada decisão de agente gera um trace auditável: input, contexto RAG, raciocínio, output, custo. Dashboard em tempo real com ROI por operação.
