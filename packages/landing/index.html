O QUE É

Exatamente o que um time de engenheiros faria.
Mas em 12 segundos, não 12 dias.

Paganini é um AIOS — AI Operating System. Cada agente é código Python com SOUL (personalidade + competências + limites definidos em arquivo), memória persistente, guardrails, e a capacidade de spawnar sub-agentes para tarefas recursivas. Não é LLM wrapper. É arquitetura de engenharia.

ARCHITECTURE STACK
LAYER 4 Skills Marketplace
Skills publicadas por terceiros. Versionadas, com dependências, SHA256 integrity. Monetizáveis. Composable como npm packages.
LAYER 3 Agent Fleet
9+ agentes com SOULs. Cada um lê, entende, decide, executa e delega. Spawnam sub-agentes recursivamente. Orquestrados pelo Cognitive Router.
LAYER 2 Cognitive Kernel
Moltis runtime + Cognitive Router (intent/domain/confidence) + Hybrid RAG (embeddings + BM25 + knowledge graph) + MetaClaw (self-learning) + RTK (token compression).
LAYER 1 Infrastructure
Python 3.12 + Docker + ChromaDB + pgvector + FastAPI. Deploy: Docker Compose (on-prem) ou cloud (managed). CLI-first.

RECURSIVIDADE

Agentes que spawnam agentes.
Cada nível adiciona contexto e precisão.

Quando um agente recebe uma tarefa complexa, ele não tenta resolver sozinho. Ele decompõe em sub-tarefas e spawna sub-agentes especializados. Cada sub-agente herda contexto, tem acesso à memória do pai, e retorna resultados que são agregados. Isso não é teórico — é como funciona na prática, todos os dias.

SKILLS MARKETPLACE

Skills são pacotes de conhecimento
que agentes instalam, usam e monetizam.

Uma skill é um diretório com metadata.json (versão, tipo, dependências, capabilities). O loader resolve a árvore, gera lockfiles com SHA256, e controla context budget via progressive loading. Skills podem ser publicadas no marketplace — outros fundos instalam e pagam por uso.

ANATOMY OF A SKILL
fidc-compliance-pack/ ├── metadata.json │ name: "fidc-compliance" │ version: "1.2.0" │ type: "specialist" │ price: "R$ 2.400/mês" │ depends: ["fidc-rules-base@^1.0"] │ capabilities: │ - check_eligibility │ - check_concentration │ - check_covenant │ - check_pld_aml │ - generate_compliance_report ├── SKILL.md # Knowledge (LLM reads this) ├── guardrails.py # Executable code ├── templates/ # Report templates └── skill.lock # SHA256 integrity
SKILL TYPES
ABSTRACT Base Knowledge
Regulamentação, normas, definições. Não executa — é referência para outros skills. Ex: fidc-rules-base, bacen-norms
SPECIALIST Domain Expert
Executa uma competência específica. Depende de um abstract. Ex: compliance-agent, pricing-agent
ORCHESTRATOR Coordinator
Compõe specialists em pipelines. Resolve dependências, gerencia context budget, coordena execução. Ex: fidc-orchestrator
