Modelo de precificação recomendado: SaaS + usage.
• Base mensal (infra + suporte + SLA): R$5.000-15.000/mês por fundo
• Usage (excedente): R$0.80-2.00 por operação FIDC acima da franquia
• BYOK discount: -30% se cliente trouxer próprias API keys
• Franquia inclusa: POC = 5K ops, Prod = 25K ops, Enterprise = 100K ops
⚠ Custos ocultos a considerar no preço:
• Suporte técnico (~20h/mês = R$5.000 em custo humano)
• Onboarding + customização de agentes (~40h one-time = R$10.000)
• Atualizações de compliance/regulatório (CVM, BCB) = contínuo
• Manutenção de corpus RAG (ingestão de novos docs) = contínuo
• Reserva para picos de uso (Black Friday de FIDCs = safra de duplicatas)

Conclusão: O custo base de rodar Paganini AIOS em EKS produção é ~$500/mês (R$2.600). O LLM é 38% — e escala linearmente com uso. Com smart routing (MetaClaw), o custo por operação FIDC cai de $0.45 (modelo único) para $0.04-0.19 (routing inteligente).

Para vender a R$8K/mês/fundo (plano Professional da proposta open-core), o custo real é R$2.600-4.000. Margem de 100-200%.

O EKS é mais caro que EC2 puro (~$300 de overhead por cluster), mas entrega: auto-scaling, rolling deploys, health checks, HA, e — crucialmente — isolamento por namespace (cada fundo do cliente roda isolado).
