Paganini vs. Broadcast

Mapeamento completo do portfolio Broadcast (Grupo Estado) e posicionamento do Paganini Intelligence Platform como alternativa AI-native para o mercado financeiro brasileiro.

O que é o Broadcast

Grupo Estado / Agência Estado — fundada em 1991. ~1.000 funcionários. Maior agência de notícias financeiras do Brasil.
Plataforma 01

Broadcast+

Terminal financeiro principal. Concorre com Bloomberg/Refinitiv no mercado brasileiro.

  • Notícias em tempo real (redação própria ~200 jornalistas)
  • Cotações B3, internacionais, câmbio, commodities
  • Análise técnica com gráficos intraday
  • Link Excel (add-in) para dados em streaming
  • Módulo de negociação eletrônica (PQO B3)
  • Precificação de opções, multi-leg
  • Indicadores macroeconômicos + séries históricas
  • App mobile para assinantes
Preço: ~R$500-2.000/mês (estimativa por consulta)
Plataforma 02

Broadcast Fundos

Análise de fundos de investimento. 3 versões: Light, Investidor, AA&AI.

  • Screener de fundos (dezenas de milhares ativos/inativos)
  • Fronteira eficiente, Sharpe, VaR, tracking error
  • Gráficos e matrizes customizáveis
  • Relatórios PDF/Excel com marca do cliente
  • Composição de carteiras comparativa
  • Aderente CVM 175
Preço: R$200-800/mês (estimativa)
Solução 03

Broadcast Datafeed

API de dados em tempo real para integração em sistemas próprios.

  • Dados B3 em baixa latência
  • Cotações, book, negócios, índices
  • Formato FIX/JSON/WebSocket
Preço: R$2.000-10.000/mês (por volume)
Solução 04

Broadcast Quant

Dados quantitativos para modelos e backtesting.

  • Séries históricas limpas
  • Eventos corporativos ajustados
  • API para consumo programático
Preço: sob consulta
Solução 05

Broadcast Radar

Monitoramento de notícias e menções por empresa/setor.

  • Alertas customizados
  • Clipping automatizado
  • Análise de sentimento (básica)
Preço: R$200-500/mês
Solução 06

Broadcast IA

Módulo de inteligência artificial integrado ao Broadcast+.

  • Análise de ativos globais
  • Resumos automatizados
  • Chatbot para consultas de mercado
Lançamento recente. Funcionalidade limitada.
Verticals

Broadcast Agro / Político / Energia

Terminais setoriais especializados.

  • Agro: commodities agrícolas, câmbio rural
  • Político: legislação, lobby tracking
  • Energia: mercado livre, PLD, contratos
White-label disponível para parceiros
Embeddable

Widgets / Ticker / Wallboard / OTC

Componentes para embed em sites e aplicações de terceiros.

  • Ticker de cotações para portais
  • Wallboard para trading desks
  • Widgets customizáveis (iframe/JS)
  • OTC: balcão organizado
Modelo de licenciamento por site/instância

Fraquezas do Broadcast

Onde Paganini ataca.
Fraqueza #1
AI é bolt-on
O módulo IA do Broadcast é um add-on recente sobre uma plataforma de 30+ anos. Não é AI-native. Paganini foi construído de AI para fora — cada decisão passa por LLM, cada output tem trace.
Fraqueza #2
Sem automação FIDC
Broadcast Fundos analisa fundos de forma passiva (screener, gráficos). Não executa guardrails, não classifica risco automaticamente, não faz DD. Tudo é manual.
Fraqueza #3
Preço legacy
Modelo de assinatura mensal fixa, contrato anual, preço por consulta. Zero pay-per-use. Zero BYOK. Cliente paga pelo terminal inteiro mesmo se usa 10% das features.
Fraqueza #4
Cloud-only
Sem opção on-premise. Dados do fundo passam pelos servidores do Broadcast. Para FIDCs com dados sensíveis de cedentes (PLD/AML, CPF/CNPJ), isso é um blocker de compliance.
Fraqueza #5
Sem customização profunda
Broadcast é one-size-fits-all. Mesmo regras de negócio, mesmos relatórios. Paganini permite agentes custom por fundo, regras por cedente, corpus isolado por cliente.
Fraqueza #6
Legacy tech stack
30+ anos de código legado. Interface Desktop-era. Sem API moderna (REST/GraphQL). Integrações via FIX protocol e add-in Excel — não cloud-native.
