## LLM Providers (BYOK — Customer Responsibility)

Paganini uses BYOK (Bring Your Own Key). The customer provides their own API key.
No LLM provider SDK is bundled; calls go through LiteLLM (MIT).

| Provider | Commercial Use | Output Ownership | Data Training |
|----------|---------------|-----------------|---------------|
| OpenAI API | ✅ Allowed | Customer owns output | API data not used for training |
| Google Gemini (Paid) | ✅ Allowed | Customer owns output | Paid tier: not used for training |
| Google Gemini (Free) | ✅ Allowed | Customer owns output | ⚠️ May be used for training |
| Anthropic Claude | ✅ Allowed | Customer owns output | API data not used for training |
| Ollama (Local) | ✅ Allowed | Customer owns output | Fully local, no external calls |
